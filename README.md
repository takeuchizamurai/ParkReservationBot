# 都立公園テニスコート抽選申込みBot

都立公園スポーツレクリエーション予約システムへの抽選申込みを自動化するツールです。

---

## 事前準備

### 1. Docker Desktop をインストールする

以下のページからダウンロードしてインストールしてください。

👉 https://www.docker.com/products/docker-desktop/

インストール後、Docker Desktop を起動しておいてください。

---

### 2. 必要なファイルをダウンロードする

以下の3つのファイルをダウンロードして、**同じフォルダ**に置いてください。

```
任意のフォルダ/
├── Dockerfile
├── docker-compose.yml
└── requirements.txt
```

---

### 3. account.csv を用意して同じフォルダに置く

CSVファイルは以下の形式で作成してください。

```
任意のフォルダ/
├── Dockerfile
├── docker-compose.yml
├── requirements.txt
└── account.csv   ← ここに置く
```

**account.csv の列の意味：**

| 列名 | 内容 | 例 |
|------|------|----|
| user_id | 利用者番号 | 10000000 |
| password | パスワード | ABCDEFG |
| user_name | 氏名 | 苗字 太郎 |
| expire_date | 有効期限 | 2026/03/30 |
| park_name1 | 1件目の公園名 | 猿江恩賜公園 |
| target_date1 | 1件目の申込み日（yyyy/mm/dd形式） | 2026/05/02 |
| target_time1 | 1件目の時間帯（開始時刻の数字） | 13 |
| park_name2 | 2件目の公園名 | 猿江恩賜公園 |
| target_date2 | 2件目の申込み日（yyyy/mm/dd形式） | 2026/05/04 |
| target_time2 | 2件目の時間帯（開始時刻の数字） | 15 |

**時間帯の数字について：**

| 数字 | 時間帯 |
|------|--------|
| 9 | 9時〜11時 |
| 11 | 11時〜13時 |
| 13 | 13時〜15時 |
| 15 | 15時〜17時 |
| 17 | 17時〜19時 |
| 19 | 19時〜21時 |

複数アカウントで申し込む場合は、1行に1アカウントずつ追加してください。

---

### 4. Dockerイメージをビルドする（初回のみ）

コマンドプロンプト（cmd）を開いて、ファイルを置いたフォルダへ移動します。

```cmd
cd C:\Users\あなたのユーザー名\ファイルを置いたフォルダ名
```

以下のコマンドを実行してください（初回のみ、数分かかります）。

```cmd
docker-compose create --build
```

`✔ Container reservation-bot Created` のように表示されれば成功です。

---

## 実行方法

### Docker Desktop から起動する

1. Docker Desktop を開く
2. 左のメニューから **Containers** を選択
3. `reservation-bot` を見つけて ▶ ボタン（Start）を押す
4. ログが流れて自動的に申込みが実行されます

### ブラウザの操作画面を確認する（reCAPTCHA手動操作用）

実行中のブラウザの画面をリアルタイムで確認したい場合は、PCのブラウザ（ChromeやEdgeなど）で以下のURLを開いてください。

```
http://localhost:6080/vnc_auto.html
```

自動的に接続され、コンテナ内で動いているブラウザの画面が表示されます。

---

## 注意事項

- 実行前に account.csv の内容が正しいか確認してください
- 申込み期間外の日付を指定するとエラーになります
- 実行中にブラウザが自動で動くことがあります（正常な動作です）
- ログや申込み完了のスクリーンショットは `screenshots/` フォルダに保存されます
